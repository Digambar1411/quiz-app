# NeoQuiz Web App

## Quick Start

Preview Link : https://my-neoquiz.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/Digambar1411/quiz-app.git
```

## Features Available.

- Home Page with all Quiz Categories
- Rules page
- Question page
- Result page
- Authentication
  - Login
  - Sigup
  - Guest Login
- Resposive Web app

## Tech Stack

**Client:** React, Context API, React Router v6, Neo UI Component library.

**Server:** Mockbee.
